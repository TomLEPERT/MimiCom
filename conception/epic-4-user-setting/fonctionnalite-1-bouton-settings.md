# Fonctionnalité 1 - Bouton "settings" dans le menu

## Description
Le bouton d'accès aux paramètres est un point clef de la sécurité de l'user experience de l'application MimiCom.
Il permet l'accès à la page des paramètres ou l'utilisateur pourra y modifier email, clef API de l'adresse email, mot de passe du compte et thème de l'application.

## Objectif
- Permettre à l'utilisateur d'accéder facilement à la page où il pourra changer ses paramètres

## Scénario
Gigi se connecte sur MimiCom.
Il veut accéder aux paramètres de son compte pour y changer le mot de passe.
Gigi clique sur le bouton "paramètres" dans le menu de l'application.
Gigi accède à la page "paramètres".

## User Stories
### US. Bouton "paramètres"
En tant qu'utilisateur authentifié, j'arrive sur la page principale de Mimicom et veut accéder aux paramètres du compte.

**Requis :**
- Bouton "paramètres" dans le menu renvoyant vers la page "paramètres"
