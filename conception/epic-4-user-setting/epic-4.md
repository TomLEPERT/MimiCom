# EPIC 4 - User settings

## Description

L'epic 3 correspond aux fonctionalités de paramétrage de Mimicom par l'utilisateur.
A l'intérieur, il aura la possibilité de modifier des informations liées à son profil (mail associé, clef API de l'adresse mail, mot de passe) ainsi qu'à l'expérience utilisateur de l'application (theme).

Cet EPIC est un point important pour la sécurité de l'application Mimicom, et doit permettre à l'utilisateur de modifier ses paramètres aisément.


## Objectifs
- Accéder aux options de paramétrage aisément
- Modifier l'adresse mail et le mot de passe
- Changer le thème du site


## Scénario principal

Un collaborateur de Gigi vient de quitter l'association. Gigi a besoin de modifier le mot de passe de MimiCom pour des raisons de sécurité.
Gigi se connecte sur MimiCom.
Il clique sur "Paramètres" dans le menu de MimiCom.
Là, les options de modifier l'email, le mot de passe ou le thème de l'application lui sont proposés.
Il change le mot de passe en inscrivant son précédent mot de passe puis saisit 2 fois le nouveau mot de passe (dans l'éventualité d'une faute de saisie).
Gigi valide l'opération et son mot de passe est désormais bel et bien modifié.


## Liste des fonctionnalités

Fonctionnalité 1 - Bouton "settings" dans le menu
- US1 - Bouton

Fonctionnalité 2 - Page paramètre
- US1 - Option de changement de l'adresse mail liée au compte
- US2 - Option de changement de la clef API du mail
- US3 - Option de changement du mot de passe du compte
- US4 - Option de changement du thème de l'application
