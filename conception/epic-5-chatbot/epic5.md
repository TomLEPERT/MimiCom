# EPIC 1 – Chatbot

## Description

Ce chatbot servira de point d'entrée pour la gestion des mailing.
Il aura pour rôle d'automatiser les réponses, d'assister les utilisateurs dans leurs démarche et de personaliser les mails en fonction de la cible
L'outil utilisera le traitement du langage naturel (NLP) pour comprendre les intentions des utilisateurs au-delà de simples mots-clés.


## Objectifs

- Automatiser le traitement des mailing
- Optimisation du temps des utilisateurs : Permettre aux équipes se concentrer sur d'autres tâches
 

## Scénario principal

L'utilisateur ouvre le widget de chat.
Il saisit une phrase, le bot identifie l'utilisateur via son token de session.
Il interroge l'API GET /orders/last
Puis le bot affiche la réponse
